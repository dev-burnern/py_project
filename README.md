## 1. 개요
본 프로젝트 **‘사랑의 큐피트’**는 카카오톡 대화를 기반으로  
상대방의 **호감도(감정 성향), 대화 패턴, 주요 키워드, 활발한 시간대** 등을 분석하여  
사용자에게 시각적·해석형 결과를 제공하는 **AI 기반 대화 분석 시스템**입니다.

사용자는 카카오톡에서  
- **대화를 복사·붙여넣기**, 또는  
- **텍스트(.txt) 파일 업로드**  

를 통해 손쉽게 데이터를 입력할 수 있습니다.

정규표현식 기반 파싱 엔진을 통해  
대화를 날짜/시간/발화자/본문으로 구조화하며,  
이후 pandas와 자연어 처리 기술(Kiwi, TF-IDF 등)을 활용하여  
대화의 특성을 **정량적·정성적으로 분석**합니다.

## 2. 주요 기능

### 2-1 텍스트 입력 기능
- 카카오톡 대화를 직접 붙여넣어 분석 가능  
- `.txt` 파일 불러오기 지원  
- 입력된 데이터를 자동 정제 후 분석에 활용  

---

### 2-2 대화 파싱 기능
- **모바일/PC 카카오톡 형식 자동 감지**  
- 정규표현식(Regex) 기반 날짜/시간/발화자/메시지 분리  
- 멀티라인(줄바꿈) 메시지 자동 병합  
- pandas DataFrame으로 변환하여 분석 기반 데이터 생성  

예시:
오전 10:23, 홍길동 : 안녕ㅋㅋ
[오후 1:23] 김철수 : 뭐해?

yaml
코드 복사

---

### 2-3 참여자별 대화 기여도 분석
- 발화자별 메시지 개수 집계  
- 전체 메시지 대비 기여율 산출  
- **막대그래프(Bar Chart)** 형태로 시각화  

예시:
- 홍길동: 120개 (40%)  
- 김철수: 90개 (30%)  
- 이영희: 60개 (20%)

---

### 2-4 키워드 분석 기능
- 한국어 형태소 분석기 **Kiwi**로 명사 추출  
- Scikit-learn CountVectorizer로 단어 빈도 계산  
- 상위 키워드 Top N 리스트 제공  
- **워드클라우드 이미지 자동 생성**  

예시:
- 과제(35회), 발표(27회), 교수님(15회)

---

### 2-5 호감도 분석 ― 사랑의 큐피트 엔진
키워드 기반 감정·관심도 추론 기능 제공:

- 애정 표현 / 칭찬 / 만남·연락 유도 단어 감지  
- 거부·부정 단어는 높은 감점  
- 최종 출력:
  - **호감 점수 (0~100)**  
  - **호감도 라벨** (예: "`호감 있는 편 💗`")  
  - **한 줄 분석 요약**  
  - **자세한 해석 요약문**  

예시 요약:
> “친근한 농담과 만남 제안이 자연스럽게 오가며 호감도가 높은 편이에요!”

---

### 2-6 대화 시간대 분석
- 메시지가 가장 활발하게 오간 시간대를 전체(0~23시) 기준으로 분석  
- **라인차트(Line Chart)**로 가독성 있게 시각화  
- 상대방의 활동 패턴 파악에 활용 가능  

---

### 2-7 결과 리포트 출력 UI
다음 요소를 화면에서 시각적으로 제공:

- 호감 점수 + 라벨 + 요약  
- 참여자 대화량 리스트 & 그래프  
- 키워드 Top 5  
- 워드클라우드 이미지  
- 시간대별 메시지 빈도 라인차트  

추가 기능(확장 가능):
- PDF 보고서 저장  
- 워드클라우드/그래프 이미지 다운  
- “내 메시지 / 상대 메시지 분리 분석” 기능

## 4. 사용 라이브러리
