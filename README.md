## 1. 개요
본 프로젝트는 카카오톡 대화방의 텍스트 데이터를 입력받아
참여자별 발화 기여도, 대화 키워드, 주요 주제를 분석하여
사용자에게 분석 결과를 시각적으로 제공하는 프로그램이다.

카카오톡 대화를 손쉽게 복사·붙여넣기 하거나
텍스트 파일 형식으로 불러올 수 있으며,
정규표현식 기반 파싱을 통해 데이터 구조화 후
pandas, 자연어 처리 기법 등을 활용하여
대화 특성을 정량적으로 파악할 수 있도록 한다.
## 2. 주요 기능
1. 텍스트 입력 기능

카카오톡 대화 내용을 복사하여 프로그램에 붙여넣기
또는 .txt 파일 불러오기 기능 제공
(사용자가 편리하게 데이터 입력 가능)

2. 대화 파싱 기능

•카카오톡 대화 형식을 자동 인식하여 날짜/시간, 발화자, 메시지 내용으로 분리
•정규표현식(Regex)을 활용하여 데이터 구조화
•pandas DataFrame 형태로 저장하여 이후 분석에 활용
•카카오톡 데이터 예시
 2025.11.29 오전 10:23, 홍길동 : 안녕ㅋㅋ

3. 참여자별 대화 기여도 분석

•사람별 메시지 개수 집계
•전체 메시지 대비 발화 기여율(%) 계산
예시 결과

•홍길동: 120개 (40%)
•김철수: 90개 (30%)
•이영희: 60개 (20%)
➡ 발화량 순위 막대그래프로 시각화하여 출력

4. 키워드(단어) 분석 기능

•메시지 텍스트 전처리 후
•scikit-learn의 TF-IDF 기반 키워드 추출 적용
•단어 빈도수 집계 후 상위 N개(예: 20개) 키워드 출력
예시
•과제(35회), 발표(27회), 교수님(15회)
➡ 워드클라우드(WordCloud) 시각화 옵션 제공

5. 대화 주제 자동 추론
•상위 키워드 조합 기반으로 대화 주제 문장 생성
•간단한 룰 기반 카테고리 분류 방식 적용

예시

•상위 키워드: 과제, 발표, 교수님
→ “학업/과제 관련 대화 중심”
•상위 키워드: 치킨, 약속, 시간
→ “모임/식사 관련 대화 중심”

6. 결과 리포트 출력
•화면 구성 요소
•발화 TOP 사용자 순위 (표/그래프)
•키워드 TOP N 리스트
•워드클라우드 이미지(선택)
•대화 주제 요약 문장(1~2줄)
•확장 기능(선택)
•분석 보고서 PDF 저장 지원
•이미지(그래프/워드클라우드) 저장 가능
