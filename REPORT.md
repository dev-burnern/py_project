# 프로젝트 보고서: 카카오톡 대화 분석 PC 애플리케이션

## 1. 주제
본 프로젝트는 카카오톡 대화 내보내기 파일을 분석하여 대화 참여자들의 발화량, 주요 키워드, 시간대별 대화 분포 등을 시각화하고, 텍스트 마이닝을 통해 대화의 분위기와 호감도를 추론해주는 **PC 설치형 애플리케이션**입니다. 사용자가 PC에서 프로그램을 실행하여 카카오톡 대화 내용을 입력하면, 로컬 서버에서 자연어 처리(NLP) 과정을 거쳐 다양한 인사이트를 제공합니다.

## 2. 시스템 구조
본 시스템은 프론트엔드(React)와 백엔드(Python Bottle)로 개발되었으며, **PyInstaller를 통해 하나의 실행 파일(.exe)로 패키징된 PC 애플리케이션**입니다.

### 시스템 아키텍처
1.  **Application Shell**: PyInstaller로 빌드된 Windows 실행 파일입니다. 내장된 Python 런타임 위에서 백엔드 서버를 구동합니다.
2.  **Backend (Local Server)**: 사용자 PC 로컬 환경에서 Bottle 웹 서버가 실행되어 데이터 분석 로직을 처리합니다.
3.  **Frontend (UI)**: React로 빌드된 정적 리소스가 로컬 서버를 통해 제공되며, 사용자의 웹 브라우저를 통해 인터페이스를 표시합니다.
4.  **Data Processing**:
    *   **Parser**: 카카오톡 대화 텍스트를 정형 데이터(DataFrame)로 변환합니다.
    *   **Analysis Engine**: Kiwi(형태소 분석기)와 Scikit-learn(단어 빈도 분석)을 사용하여 키워드 추출 및 호감도 분석을 수행합니다.

### 데이터 흐름
1.  사용자가 `.exe` 파일을 실행하여 로컬 서버 구동
2.  웹 브라우저가 자동으로 열리거나 사용자가 접속하여 UI 로드
3.  사용자가 프론트엔드에서 텍스트 입력 후 '분석하기' 버튼 클릭
4.  프론트엔드에서 `POST /api/analyze_text` (Localhost) API 호출
5.  백엔드에서 텍스트 수신 후 임시 파일 저장 및 파싱
6.  분석 모듈이 참여율, 키워드, 호감도, 시간대 분포 계산
7.  JSON 형식으로 분석 결과 반환
8.  프론트엔드에서 결과를 시각적으로 렌더링

## 3. 사용 라이브러리 및 API 정보

### 3.1 Backend (Python)
*   **Bottle**: 가볍고 빠른 웹 서버 구축을 위한 마이크로 웹 프레임워크입니다. 로컬 API 서버 역할을 담당합니다.
*   **PyInstaller**: Python 애플리케이션과 모든 의존성(React 빌드 파일 포함)을 단일 실행 파일(.exe)로 패키징하는 데 사용됩니다.
*   **Pandas**: 대화 데이터를 효율적으로 처리하고 분석하기 위한 데이터 조작 라이브러리입니다.
*   **KiwiPiePy**: 한국어 형태소 분석을 위한 라이브러리로, 명사 추출 및 텍스트 토큰화에 사용됩니다.
*   **Scikit-learn**: `CountVectorizer`를 사용하여 텍스트 데이터에서 단어 빈도를 계산하고 핵심 키워드를 추출하는 데 사용됩니다.

### 3.2 Frontend (JavaScript)
*   **React**: 사용자 인터페이스를 구축하기 위한 라이브러리입니다.
*   **Vite**: 빠른 개발 환경과 빌드를 위한 프론트엔드 빌드 도구입니다.

### 3.3 API 명세
| Method | Endpoint | 설명 |
| :--- | :--- | :--- |
| `POST` | `/api/analyze_text` | 사용자가 입력한 카카오톡 대화 텍스트를 분석하여 결과를 반환합니다. |
| `GET` | `/` | 프론트엔드 메인 페이지(`index.html`)를 서빙합니다. |
| `GET` | `/assets/<filepath>` | 프론트엔드 정적 리소스(JS, CSS, 이미지 등)를 제공합니다. |

## 4. 실행 화면

*   PC 애플리케이션 실행 시 나타나는 초기 화면입니다.

### 4.2 분석 결과 화면
[분석 결과 화면]
*   **참여율 분석**: 대화 참여자별 발화 비중을 보여줍니다.
*   **키워드 분석**: 대화에서 가장 많이 사용된 핵심 단어들을 보여줍니다.
*   **호감도 분석**: 대화 내용을 바탕으로 추론된 호감도 점수와 요약을 제공합니다.
*   **시간대별 분석**: 하루 중 대화가 가장 활발한 시간대를 그래프로 보여줍니다.
